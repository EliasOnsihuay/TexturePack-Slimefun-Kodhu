# TexturePack Slimefun Kodhu Bedrock Preview

Version: `v0.1.1`

This is a Minecraft Bedrock Edition preview/import package based on TexturePack Slimefun Kodhu.

## Important

This is not a full Java-to-Bedrock conversion.

Minecraft Bedrock Edition does not support Java resource pack features such as:

- Java item model overrides
- Java `pack.mcmeta` overlays
- OptiFine/CIT Resewn `.properties`
- Slimefun Java custom item behavior
- ItemsAdder Java resource pack generation behavior

## What This Package Includes

- Bedrock `manifest.json`
- Pack icon
- Credits, attribution, notice, and license files
- A small curated set of representative Slimefun/addon textures for branding/reference

## Install

Open `TexturePack-Slimefun-Kodhu-Bedrock-Preview-v0.1.1.mcpack` with Minecraft Bedrock Edition.

## Recommended Use

Use the Java Edition ZIP for the real Slimefun experience:

- `TexturePack-Slimefun-Kodhu-v0.1.1.zip`

Use this Bedrock package only as an experimental starter or branding/resource reference.
