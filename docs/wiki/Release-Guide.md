# Release Guide

Before publishing:

1. Update `VERSION`.
2. Update `pack.mcmeta`.
3. Update `README.md`.
4. Update `CHANGELOG.md`.
5. Regenerate Java ZIP.
6. Test Java ZIP with 7-Zip.
7. Regenerate Bedrock Preview `.mcpack` if needed.
8. Test `.mcpack` with 7-Zip.
9. Push docs to GitHub.
10. Create GitHub release.
11. Upload Java ZIP and Bedrock Preview `.mcpack`.

See [../../RELEASE_CHECKLIST.md](../../RELEASE_CHECKLIST.md).
