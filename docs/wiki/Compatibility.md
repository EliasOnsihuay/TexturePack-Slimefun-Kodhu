# Compatibility

## Java Edition

Primary target:

- Java Edition Slimefun servers
- Best-effort client support from 1.16+ to modern versions

Pack metadata:

- `pack_format: 6`
- `supported_formats: 6-500`

## ViaVersion

ViaVersion helps clients join servers across versions. It does not fully convert resource-pack formats, custom item models, CIT files, or overlays.

The player's client still controls which resource-pack features can work.

## CIT / Custom Items

Some custom item textures may depend on:

- OptiFine CIT
- CIT Resewn
- Minecraft custom model behavior
- Server/plugin item metadata

## Slimefun

Slimefun is a Java server plugin. This pack is built around Java Slimefun behavior.
