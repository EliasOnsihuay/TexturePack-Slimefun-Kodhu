# Installation

## Java Personal Install

1. Download `TexturePack-Slimefun-Kodhu-v0.1.1.zip`.
2. Move the ZIP to `.minecraft/resourcepacks`.
3. Start Minecraft Java Edition.
4. Open Options > Resource Packs.
5. Enable the pack.
6. Place it above other Slimefun packs.

## Java Server Install

Use a direct resource-pack host, then add the URL to `server.properties`.

See [Server Hosting](Server-Hosting.md).

## Bedrock Install

Import `TexturePack-Slimefun-Kodhu-Bedrock-Preview-v0.1.1.mcpack`.

This is only a preview package. It does not provide full Slimefun Java item model behavior.
