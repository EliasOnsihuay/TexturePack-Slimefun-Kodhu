# Bedrock Preview

The Bedrock package is not a full Bedrock port.

File:

```text
TexturePack-Slimefun-Kodhu-Bedrock-Preview-v0.1.1.mcpack
```

## What Works

- Bedrock manifest
- Pack icon
- Import test
- A small set of representative textures
- Credits and license files

## What Does Not Fully Work

- Java custom item model overrides
- Java `pack.mcmeta`
- Java overlays
- OptiFine/CIT Resewn `.properties`
- Slimefun Java custom item behavior

## Full Bedrock Support

Full Bedrock support would require a separate Bedrock addon or a Geyser/custom item mapping workflow.
