# FAQ

## Does this replace Slimefun?

No. This is only a resource pack. You still need the Slimefun plugin on a Java server.

## Does it work on Bedrock?

Only as a preview/import package. Full Slimefun Java model behavior does not work on Bedrock by itself.

## Does ViaVersion make it compatible with every client?

No. ViaVersion helps players join servers, but the client still decides resource-pack support.

## Should I host the pack on GitHub for my server?

Use GitHub for releases and documentation. For `server.properties`, use a direct ZIP host such as MCPacks.

## Why do some textures not appear?

Possible reasons:

- Missing OptiFine/CIT Resewn support
- Wrong resource pack order
- Unsupported client version
- Server item metadata differs from the pack rules
- Addon item changed names or IDs

## Can I use this on a public server?

Yes, as long as credits and license/notice files are preserved.
