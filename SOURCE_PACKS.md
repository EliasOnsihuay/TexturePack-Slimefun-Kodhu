# Source Packs

TexturePack Slimefun Kodhu v0.1 was assembled from the following local source archives:

| Source archive | Role |
|---|---|
| `Slimefun 1.21.5+ v2.3.zip` | Main modern base |
| `Slimefun-ResourcePack.zip` | Additional assets, addon namespaces, and overlays |
| `Slimefun-Resourcepack-main.zip` | Project source reference and generated resource pack assets |
| `To Infinity 1.19.4+ (4-3-26).zip` | Additional CIT/resource pack assets and older compatibility content |

## Merge Policy

The newest Slimefun-compatible pack was used as the base. Missing files from the other packs were added. When duplicate paths existed, the newer modern version was preferred to reduce compatibility problems.
