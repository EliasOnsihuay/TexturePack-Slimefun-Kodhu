# Contributing

Thanks for helping improve TexturePack Slimefun Kodhu.

## What Helps

- Missing texture reports
- Broken model reports
- Better addon credits
- Compatibility notes for specific Minecraft versions
- Screenshots showing incorrect items
- Improved Bedrock/Geyser mapping research

## Good Bug Report

Include:

- Minecraft client version
- Server version
- Slimefun version/build
- Addons installed
- Whether you use ViaVersion
- Whether you use OptiFine or CIT Resewn
- Screenshot or item name
- Resource pack order

## Pull Requests

Keep changes focused. Do not remove credits or license files.

If you add third-party assets, include source and author information in `ATTRIBUTION.md` and `CREDITS.md`.
